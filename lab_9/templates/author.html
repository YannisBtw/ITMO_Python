Это учебный проект по дисциплине "Программирование на Python" Ломаченко Яна (505115). Цель приложения — продемонстрировать работу с HTTP-сервером, архитектурой MVC, шаблонизатором Jinja2 и интеграцией с внешним API (курсы валют ЦБ РФ).

В рамках проекта реализованы:

  • модели предметной области (Author, App, User, Currency, UserCurrency);
  • получение курсов валют через API ЦБ РФ;
  • логирование запросов к API;
  • рендеринг HTML-страниц через Jinja2;
  • маршрутизация запросов на основе HTTPServer.
